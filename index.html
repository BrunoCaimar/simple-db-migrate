simple-db-migrate by guilhermechapiewski

simple-db-migrate is a database versioning and migration tool inspired on Rails Migrations.

This tool helps you refactor, manage and track your database schema.

The main difference is that Rails migrations are intended to be used on Ruby projects while simple-db-migrate makes it possible to have migrations with any kind project or language (Java, Python, Ruby, PHP, whatever).

This is possible because simple-db-migrate uses database's DDL (Data Definition Language) to do the database operations, while Rails Migrations are written in a Ruby DSL.
